# linkedin-posts-assets

Хостинг медиа для постов LinkedIn через GitHub Pages. Файлы отдаются по стабильным публичным ссылкам, которые Buffer тянет в момент публикации.

- `images/` — картинки постов (PNG, 1080×1080 и т.п.)
- `carousels/` — карусели в PDF (1080×1350)

Базовый URL: https://cpa-instructor.github.io/linkedin-posts-assets/
